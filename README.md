# LaravelBlog

A Laravel Blog application.

Functions:

	Welcome page with recent posts,
	Contact us page with a email form,
	Blog page that listing all posts with pagination,
	About us page,

	Create Posts,
	Edit Posts,
	Delete Posts,

	Create Tags,
	Delete Tags,
	Edit Tags,
	Add many Tags To a post,

	Create Categories,
	Edit Categories,
	Delete Categories,
	Add a Post to a Categorie,

	Sign up users,
	Login/logout users,
	Reset user password,
